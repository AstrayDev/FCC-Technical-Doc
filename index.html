The little I know of C#

The simplest thing I can think to start with is basic text, which to write in C# is Console.Write(); or Console.WriteLine();. Methods like this are followed by a semicolon to signify the end of the command. And the differnce between these two methods is that the first one simply writes the text enterd on the current line while the latter writes on a new line. Inside the paranthesis are where you enter either text, numbers or variables, which we'll get to in a moment. Note: Text needs to be within quotation marks, for example, Console.Write("Hoi");.
Another important aspect of C# is that it is case sensitive, meaning the "C" and "W" in Console.Write have to be capitilized otherwise it'll throw an error.

You can also assign preset values to specific keywords called variables. For example instead of writing and equation that you want to use multiple times like 42*4+23-4/8 for whatever odd reason, you can easily use this somewhere else by assigning it to a variable like so int equation = 42*4+23-4/8;. Now it is tied to equation, neato. Another Note: In C# you need to state what type of variable your creating, such as numbers need to be in a int while strings of text are in string variable types. What I've gathered for HTML, CSS

HTML & CSS, the languages I'm writing this in. I gotta say though, css is a right bitch. But I'm not gonna rant about my own skill issue, instead I'm going to attempt at putting down a small margin of what I know into text. Lets start with HTML (Hyper Text Markup Language), it is fairly simple in concept and mostly consists of "tags" such as <p> which is an opening tag for paragraphs and is ended by a closing tag such as </p>. The only differnce with closing tags is the forward slash. Making text on screen is as simple inputting text between these tags. Of course there are many more such as <a> for links, <li> for lists and <button> for well...buttons.

However, before you start customizing the page with tags and making as sick webpage, you need to lay the foundations first.
To start you need to declare the page format with <! DOCTYPE html>, and then the entire page needs to be wrapped in a <html> tag. After that you need to make a <head> tag, in which you enter your page title within <title>. To edit it with css later on you need to link it with <link>. This is a tag that doesn't need a closing tag along similar to many others I won't get into. Before the ">" you enter "rel=stylesheet" and "href=stlyes.css", the rel, as far as I know, states what type of page the link is going to or how it affects the webpage...I think. And "href" is just how you signify urls or files to link to in html. After that you need a <body> tag to enter the actual content of the page.

Now for CSS (Cascading Style Sheets), it allows you to edit the style and layout of your page to make it purty. But to edit anything you need to specify what element you changing, for example, lets say I want to edit all <p> elements, I would write p {}. Within the brackets you write the type of changes you want to make, which is a VERY large pool and I'm not even gonna start with that. To target more specific elements you need to mark them in the html page by adding either a class or an id. You can enter these like this <p class="first-paragraph"> or <p id="last-paragraph">. Note: It is recommended to use classes if you want to edit various elements the same way, such as all bold and id's for a specific element, as you can only use a single id name for one element. Targeting classes is css is just a period followed by the clas name and id's need a hashtag before the name.
